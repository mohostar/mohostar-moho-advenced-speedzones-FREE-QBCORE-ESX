# ⚡ Moho SpeedZones

[HUN] Professzionális sebességkorlátozó megoldás FiveM szerverekre. A script megakadályozza a nagy sebességű behajtást frekventált területekre, ezzel hatékonyan védi a játékosokat a "DB" (Drive-by) jellegű szabályszegésektől és a szándékos ütközésektől.

[ENG] Professional speed-limiter solution for FiveM servers. Its primary goal is to ensure road safety by preventing high-speed entry into designated areas, effectively protecting players from vehicle-based rule breaks (DB) and intentional collisions.

---

## 🚀 Why use this script? / Miért használd?

* 🛡️ **Anti-DB Protection:** Prevents players from ramming into populated areas at high speeds.
* ⚙️ **Server Stability:** Lightweight, zero-impact performance.
* 🎨 **User-Friendly:** Create and manage zones in-game with an intuitive UI.
* 🔗 **Multi-Framework:** Native support for QB-Core and ESX.

---

## 🛠️ Key Features / Főbb jellemzők

* 📍 **Dynamic Zoning:** Define speed-restricted areas in real-time.
* 💾 **Automatic Persistence:** Zones are saved to `lassitok.json` automatically.
* 👮 **Staff Security:** Server-side authorization ensures only admins can manage zones.
* ⚡ **High Performance:** Optimized for 24/7 server environments.

---

## 📦 Installation / Telepítés

1. 📥 **Download** the latest version from [Releases].
2. 📁 **Move** `moho-speedzones-qb-esx` to your `resources` folder.
3. ⚙️ **Configure** your framework in `config.lua`.
4. 📝 **Add** `ensure moho-speedzones-qb-esx` to your `server.cfg`.
5. 🔄 **Restart** server.

---

## ⌨️ Commands / Parancsok

* `/speedzones` – Open the admin management menu.

---

## 📋 Requirements / Követelmények

* 📚 **ox_lib** (Highly recommended for UI elements)
